# # Steps Involved in This Project 

**Prerequirties**

- Step 1 : Install Dependencies - **pip install boto boto3 ansible**

- Step 2 : Configuring AWS Creditinals - **aws configure**

- Step 3 : Install Required Collections - **ansible-galaxy collection install amazon.aws**


**TASK 1 : EC2 Provisioning using Loops**

- Step 1 : Create an IAM User and Generate API Keys

----

- Step 2 : Create an **Ansible YAML playbook** to automate the provisioning and management of AWS EC2 instances. You can either write your own playbook or refer to examples from official Ansible documentation

----

- Step 3 : Create an Ansible Vault file inside the group_vars directory to securely store sensitive information such as AWS API keys.

  **ansible-vault create group_vars/all/s.yml - Vault-password-file vault.pass**

----

- Step 4 : Execute the playbook and cross-verify in the AWS Management Console that three EC2 instances (2 Ubuntu and 1 CentOS) are provisioned successfully.

**TASK 2 : Passwordless Authentication**

- Step 1 : Set Permissions for the Key Pair (PEM File). Run the following command to restrict access: **chmod 400 keypairname.pem**

<img width="922" height="101" alt="image" src="https://github.com/user-attachments/assets/a28518ff-c349-4473-b4c3-492f6654a3b5" />

- Step 2 : Configure Security Group Inbound Rule for SSH Access

Edit Inbound Rules and add the following:

  - Type: SSH
  - Protocol: TCP
  - Port Range: 22
  - Source:0.0.0.0/0 (for public access) OR your IP (recommended for security)

---
